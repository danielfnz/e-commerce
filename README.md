[Ver especificação completa](https://github.com/danfma/pos-ufg-2018/wiki/Trabalho-final)

## Instalação
    $ git clone https://github.com/danielfnz/e-commerce.git
    $ cd e-commerce
    $ npm install
  
## Executando o projeto
    
    $ ng serve
